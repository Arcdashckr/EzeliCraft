# Modrinth Collection Scraper
this scrape collection and create formatted markdown file for obsidian app and discord messages
you can change collection link in `index.js`

## Usage
install node.js (and maybe npm)
open terminal (command prompt) on main folder (have index.js)
type `node index.js`
select what type content you scrape
