## 🎮 MINECRAFT MOD LISTESI (Parça 2) 🎮

### ⚡ Optimization (Performans / FPS)
🔹 **[Sodium](https://modrinth.com/mod/sodium)**
> 📝 A high-performance rendering engine replacement for Minecraft, which greatly improves frame rates and reduces micro-stutter.

🔹 **[Lithium](https://modrinth.com/mod/lithium)**
> 📝 No-compromises game logic optimization mod, useful for both single-player games and multi-player servers.

🔹 **[ImmediatelyFast](https://modrinth.com/mod/immediatelyfast)**
> 📝 Speed up immediate mode rendering in Minecraft

🔹 **[Sodium Extra](https://modrinth.com/mod/sodium-extra)**
> 📝 A Sodium addon that adds features that shouldn't be in Sodium.

🔹 **[Dynamic FPS](https://modrinth.com/mod/dynamic-fps)**
> 📝 Reduce resource usage while Minecraft is in the background, idle, or on battery.

🔹 **[Language Reload](https://modrinth.com/mod/language-reload)**
> 📝 Reduces load times and adds fallbacks for languages

🔹 **[Model Gap Fix](https://modrinth.com/mod/modelfix)**
> 📝 Fixes gaps in Block Models and Item Models

🔹 **[Noisium](https://modrinth.com/mod/noisium)**
> 📝 Optimises worldgen performance for a better gameplay experience.

🔹 **[Enhanced Block Entities](https://modrinth.com/mod/ebe)**
> 📝 Reduce block entity FPS lag with almost no compromises, and improve their visuals

🔹 **[Packet Fixer](https://modrinth.com/mod/packet-fixer)**
> 📝 A simple mod to fix various problems with packets, nbt and timeouts.

🔹 **[Let Me Despawn](https://modrinth.com/mod/lmd)**
> 📝 Improves performance by tweaking mob despawn rules. Say bye to pesky unintentional persistent mobs.

🔹 **[Fast IP Ping](https://modrinth.com/mod/fast-ip-ping)**
> 📝 Yeet the laggy reversed DNS lookup for literal IP server addresses

🔹 **[ToolTipFix](https://modrinth.com/mod/tooltipfix)**
> 📝 Fixes Tooltips from runnning off the screen.

🔹 **[Immersive Optimization](https://modrinth.com/mod/immersive-optimization)**
> 📝 A lightweight entity-tick-scheduler for doubling your TPS.

🔹 **[Sodium Extra Information](https://modrinth.com/mod/sodium-extra-information)**
> 📝 A Sodium Extra addon that adds more in-game information.

🔹 **[Staaaaaaaaaaaack (Stxck)](https://modrinth.com/mod/staaaaaaaaaaaack)**
> 📝 A mod made to merge dropped items beyond the vanilla stack limit while considering mod compatibility.

🔹 **[Server Resourcepack Checker](https://modrinth.com/mod/server-resource-pack-checker)**
> 📝 Only load server resource packs when necessary

🔹 **[Disable Portal Checks](https://modrinth.com/mod/disable-portal-checks)**
> 📝 Disable unnecessary portal checks that cause lag.

🔹 **[No Beacon Particles](https://modrinth.com/mod/no-beacon-particles)**
> 📝 Gets rid of the swirl particles from ambient status effects given by beacons and conduits. Server OR Client-side!
